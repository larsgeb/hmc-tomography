Test reference¶

Overview of available tests¶

All the modules and functions below can be accessed through hmc_tomography.Tests as follows:

from hmc_tomography.Tests import *
test_mass_matrices.test_creation() # e.g.

test_mass_matrices

A collection of tests for mass matrices.

test_mass_matrices.test_creation

Test for the creation of mass matrices.

test_mass_matrices.test_generate

Test for the generation of momenta from mass matrices.

test_mass_matrices.test_kinetic_energy

Test for the computation of kinetic energy for a given momentum.

test_mass_matrices.test_kinetic_energy_gradient

Test for the computation of kinetic energy gradient for a given momentum.

test_priors

A collection of tests for mass matrices.

test_priors.test_creation

Test for the creation of prior.

test_priors.test_generate

Test for the generation of prior samples.

test_priors.test_misfit

Test for the computation of prior misfits.

test_priors.test_gradient

Test for the computation of prior gradients.

test_priors.test_bounds

Test for correct implementation of prior bounds.

Mass matrix tests¶

A collection of tests for mass matrices.

hmc_tomography.Tests.test_mass_matrices.test_creation(mmclass: hmc_tomography.MassMatrices._AbstractMassMatrix, dimensions: int)[source]¶

Test for the creation of mass matrices.

Parameters

  • mmclass (hmc_tomography.MassMatrices._AbstractMassMatrix) – A mass matrix class.

  • dimensions (int) – Dimensions to check the mass matrix.

This test checks if we can create a given mass matrix. Using pytest, it will loop over all available subclasses of the mass matrix, with variable amount of dimensions.
