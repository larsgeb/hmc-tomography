Target objects¶

Targets or likelihood distributions for reference. Your own target class should inherit from the Target ABC.

Implementing your own targets¶

Target ABC¶

class hmc_tomography.Targets._AbstractTarget[source]¶

Abstract base class for inverse problem targets. Defines all required methods for derived classes.

full_name() → str[source]¶

Returns the full name of the target

abstract gradient(coordinates: numpy.ndarray) → numpy.ndarray[source]¶

Returns the gradient of the misfit at the given coordinates: \(∇_m \chi(m) = - ∇_m \log L(m)= - ∇_m \log p(m|d)\).

abstract misfit(coordinates: numpy.ndarray) → float[source]¶

Returns the misfit at the given coordinates. This is equal to the negative logarithm of the likelihood function: \(\chi(m) = - \log L(m)= - \log p(m|d)\).
