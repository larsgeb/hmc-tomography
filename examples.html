Examples and tutorials¶

All these tutorials are generated from Jupyter Notebooks that can be found in the project folder /examples/notebooks.

If you don’t understand a particular piece of inverse theory related to sampling, write us a message (issue) on the GitHub repo and we will try to throw a tutorial together.
